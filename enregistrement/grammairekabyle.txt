## **II. SPÉCIFICITÉS LINGUISTIQUES POINTUES DU TASAHLIT**

### A. Phonologie Remarquable

**Caractéristiques uniques**  : [lingref](http://www.lingref.com/bucld/47/BUCLD47-24.pdf)

1. **Consonants richesse exceptionnelle** (~40+ segments)
   - Haute concentration fricatives (phénomène rare)
   - Secondary articulations : pharyngealisés [tˤ], labialisés [kʷ]
   - Spirantization dialectale (Western ↔ Eastern variation)

2. **Système vocalique minimaliste** : /i/, /a/, /u/ seulement
   - Schwa [ə] épenthétique
   - Réalisations influencées par emphase

3. **Mots sans voyelles**  : [rɡl] 'close!', [tskːststː] 'you took it' [glossa-journal](https://www.glossa-journal.org/article/id/10438/)
   - Phonomène cross-linguistically rare
   - Consonants peuvent être noyau syllabique
   - Challenge ASR : syllabification non-standard

### B. Morphologie Nonconcatenative [people.linguistics.mcgill](http://people.linguistics.mcgill.ca/~mcgwpl/McGWPL/2020v26n01/2020_26_1_Ouali.pdf)

**Root-Template System** :
```
Root (3-consonants skeleton) + Vocalic melody = Surface form
f-d + i-u = ifdu 'he sleeps'
```

- Morphèmes non-contigus dans surface form
- Verbes : aorist, perfective, imperfective (aspect-driven)
- Genre/nombre : prefixes + ablaut simultané

**Implication NLP** : Standard tokenizers échouent → need morphology-aware encoding

### C. Emprunt & Contact Linguistique [langsci-press](https://langsci-press.org/catalog/view/235/1820/1846-1)

- **>20% loanwords** tous domaines sémantiques
- **Integration patterns** : integrated (a-) vs. non-integrated (l-)
- **Code-switching Amazigh-Arabic-French** fréquent en Béjaïa
- **Morphosyntactic rules differ** par type loanword

## **IV. FRAMEWORK PROPOSÉ : IMPLÉMENTATION INTÉGRÉE**

### Tier Architecture ELAN (Multi-layer)

```
Audio base
├─ Orthography (Tifinagh IRCAM)
├─ IPA Phonetic + secondary articulations (emphasis, labialization)
├─ Morphological segmentation (root + template + affixes + gemination)
├─ POS tags (UD universal)
├─ Morphosyntactic features (gender, number, aspect, tense)
├─ Syntactic dependencies
├─ Loanword flag + source language
├─ Dialect/regional marking
├─ Code-switching transitions
└─ Metadata
```

### CoNLL-U Extension Kabyle

Custom FEATS fields :
- `Root` : radical consonantiel
- `Template` : gabarit vocalique
- `Morphtype` : {concatenative, nonconcatenative, loanword}
- `Emphasis` : [+/-emphatic]
- `Labialized` : [+/-labialized]
- `Loanword_source` : {arabic, french}
- `Syllabic` : [+/-syllabic_consonant]

### Stratégie ASR Fine-tuning [arxiv](https://arxiv.org/abs/2503.18485)

**Data Mixing** (proven approach) :
- 50 hours local Kabyle + 10 hours Mozilla Common Voice + 5 hours FLEURS
- Rationale : Local-only → overfitting; multilingual mix → robustness

**Model** : OpenAI Whisper (Large)

**Evaluation** : WER by dialect + speaker robustness
