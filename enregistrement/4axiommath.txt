## 2️⃣ Axiomes techniques du Language Layer (citable + testable)

### **Axiome A1 — Langage ⇒ Transition d’état**

**Énoncé :** tout segment utile doit produire une **transition d’état** du dossier opérationnel.
**Formel :**

* Soit un état ( S_t ) (connaissance courante) et un segment ( u_t ) (utterance).
* Le Language Layer calcule :
  [
  S_{t+1} = f(S_t, u_t)
  ]
  **Test :** si ( S_{t+1}=S_t ) trop souvent → ton système “écoute” mais **ne comprend pas**.

---

### **Axiome A2 — Sens = (intention, entités, relations) + contexte**

**Énoncé :** la compréhension ne se réduit pas aux entités ; elle inclut l’intention + relations, **conditionnées par le contexte**.
**Formel :**
[
Meaning(u_t) = g(u_t, Context_t) = \langle Intent_t, Entities_t, Relations_t \rangle
]
**Test :** si le modèle extrait des entités correctes mais se trompe sur l’action → ton ( Intent_t ) / ( Relations_t ) est faible.

---

### **Axiome A3 — Action proposée + incertitude explicitée**

**Énoncé :** toute sortie doit proposer **une action** ET exposer l’**incertitude** (sinon c’est dangereux).
**Formel :**
[
Output_t = \langle Decision_t, Priority_t, Confidence_t, Evidence_t \rangle
]
**Test :** si tu n’as pas `confidence` + `evidence`, tu ne peux ni auditer ni améliorer.

---

### **Axiome A4 — Contrat stable, modèles interchangeables**

**Énoncé :** la couche sémantique doit rester stable même si tu changes ASR/LLM.
**Formel :**
[
Contract(Output) \ \text{stable} \quad \forall \ \text{Model versions}
]
**Test :** si changer de modèle casse ton pipeline → ton “Language Layer” n’existe pas vraiment (c’est juste un prompt).

### 3. Script Python simple de conversion → train.jsonl (Hugging Face ready)

```python
import json
import argparse
from pathlib import Path

def convert_to_jsonl(input_dir: str, output_file: str):
    """
    Prend un dossier de fichiers JSON (schema complet)
    Génère un train.jsonl pour QLoRA :
    {"instruction": ..., "input": ..., "output": ...}
    """
    output_lines = []
    input_path = Path(input_dir)
    
    for json_file in input_path.glob("*.json"):
        with open(json_file, "r", encoding="utf-8") as f:
            data = json.load(f)
        
        if "asr" not in data or "segments" not in data["asr"]:
            continue
        
        for seg in data["asr"]["segments"]:
            if "instruction_tuning_pair" not in seg:
                continue
                
            pair = seg["instruction_tuning_pair"]
            line = {
                "instruction": pair["instruction"],
                "input": pair["input"],
                "output": pair["output"]
            }
            output_lines.append(json.dumps(line, ensure_ascii=False))
    
    with open(output_file, "w", encoding="utf-8") as f:
        for line in output_lines:
            f.write(line + "\n")
    
    print(f"Conversion terminée : {len(output_lines)} exemples → {output_file}")

if __name__ == "__main__":
    parser = argparse.ArgumentParser()
    parser.add_argument("--input_dir", required=True, help="Dossier des JSON annotés")
    parser.add_argument("--output_file", default="train.jsonl", help="Fichier sortie")
    args = parser.parse_args()
    convert_to_jsonl(args.input_dir, args.output_file)
```

**Utilisation** :
```bash
python convert.py --input_dir ./annotated_calls/ --output_file train.jsonl
```
