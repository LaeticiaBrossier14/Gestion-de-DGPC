## 🔗 Architecture

```
User Input
    ↓
[LancerMenu.bat / CLI]
    ↓
├─→ transcribe_from_506.py (Whisper)
│   ├─→ Load audio_processed/appelle NNN.wav
│   ├─→ Transcribe (Whisper model)
│   └─→ Save to dataset/ CSV
│
├─→ transcribe_qwen3_from_506.py (Qwen3-ASR)
│   ├─→ Load audio_processed/appelle NNN.wav
│   ├─→ Transcribe (Qwen3-ASR model)
│   └─→ Save to dataset/ CSV
│
└─→ annotation_app/dgpc_annotation_local.py (Streamlit)
    ├─→ Read dataset CSV
    ├─→ Display audio + transcription
    ├─→ Call Gemini API
    ├─→ Apply Kabyle Corrector
    └─→ Export annotated CSV

Output: dataset/506_onwards_transcriptions_FINAL.csv
```

## 📞 Quick Support

| Problème | Solution |
|----------|----------|
| Pas de GPU | `python transcribe_from_506.py --use-cpu` |
| Module manquant | `pip install -r requirements.txt` |
| Fichier .wav pas trouvé | Vérifier `audio_processed/` |
| Streamlit blanche | `streamlit cache clear` |
| Slow transcription | Utiliser GPU (installer CUDA) |
